# Продвижение в Картах

Яндекс Карты — сервис, который помогает людям прокладывать маршруты, находить нужные места и решать повседневные задачи. Карточки организаций отображаются как в ответ на поисковые запросы пользователей, так и при обычном просмотре карты. Дополнительные атрибуты в карточке позволяют выделиться среди конкурентов и сделать организацию более заметной, привлекая внимание пользователей.

## Как работает

### Первые позиции в поисковой выдаче

Объявления чаще всего отображаются на первых 10 позициях поисковой выдачи. Это наиболее заметная зона, которая привлекает внимание пользователей. По статистике первые десять позиций получают до 86% всех переходов.

**Релевантные запросы**

Объявление показывается только по запросам, которые связаны с вашими услугами, товарами или сферой деятельности.

**Исключения для показа**

Объявление не отображается:
- по запросу с названием вашей организации или организации конкурентов;
- по запросу с указанием полного адреса.

**Место в выдаче**

Позиция компании зависит от ставки за клик и от того, совершит ли пользователь целевое действие (например, звонок или построение маршрута), когда ему будет показана карточка организации по конкретному запросу.

Вероятность рассчитывается индивидуально для каждого пользователя, с учетом качества услуг (рейтинг, отзывы), а также контента, из которого состоит карточка организации. Чтобы повысить шансы на показы в топовых позициях, корректно заполните рубрики карточки организации, укажите ключевые признаки, а также добавьте каталог товаров и услуг. Чем полнее и качественнее данные — тем выше вероятность привлечь ваших клиентов.

**Ограничение для сетевых компаний**

В выдаче одновременно может показываться не более пяти филиалов одной сети.

### Показ в карусели «Похожие организации»

Объявление может показываться в карусели **Похожие организации**, если поисковый запрос содержит упоминание бренда компании конкурентов.

Карусель показывается, если:
- у организации из запроса не подключено продвижение по собственному бренду;
- у организации из запроса подключено продвижение, но она не смогла победить всех участников аукциона.

![Показ в карусели «Похожие организации»](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/similar-org-mob.png)

Управлять показами по запросам с упоминанием конкретных брендов нельзя. Система автоматически подбирает организации для карусели по рубрике, размеру организации, местоположению и проводит между ними аукцион.

В Простом старте и Мастере кампаний показы в карусели подключены по умолчанию. В Единой перфоманс-кампании можно настроить показы в карусели с помощью категорий автотаргетинга.

| Условие показа карусели | Позиция в выдаче | Количество карточек |
| :--- | :--- | :--- |
| У организации из запроса не подключено продвижение по собственному бренду или в аукционе карусели победили конкуренты | 3 | 3 |
| Организация из запроса победила одного конкурента в карусели | 4 | 2 |
| Организация из запроса победила двух конкурентов в карусели | 5 | 1 |
| Организация из запроса победила всех конкурентов в карусели | Не показывается | — |

## Как работает при просмотре Карт без поиска

При просмотре Карт без ввода поискового запроса метки организаций показываются по интересам и привычкам пользователей. Метка организации с продвижением с большей вероятностью появится на карте. Название и значок будут видны при масштабировании раньше, чем у других компаний поблизости.

Метки органической выдачи и платного продвижения выглядят одинаково. Организация для показа выбирается по результатам аукциона. Оплата происходит только за клики по метке платного продвижения.

## Зеленая метка в поиске на карте

Метки компаний отображаются в Картах во время поиска. У организаций без продвижения они синего цвета. Метки компаний с продвижением в Картах — зеленого цвета, если объявление выиграло аукцион. Зеленые метки крупнее и ярче, что выделяет организацию среди остальных. Даже при уменьшении масштаба карты такая метка остается заметной, привлекая внимание пользователей и помогая потенциальным клиентам запомнить ее расположение.

Рядом с зеленой меткой могут отображаться название организации и дополнительные элементы, например рейтинг организации, время работы, информация о товарах или услугах.

Дополнительные элементы помогут привлечь внимание пользователей и сделают взаимодействие с карточкой организации еще удобнее.

## Атрибуция в WhatsApp

Если вы используете в Картах ссылку на WhatsApp, в первое сообщение от клиента будет добавлен шаблон с текстом «Обращение из Яндекс Карт. Здравствуйте! Меня заинтересовало ваше предложение».

> **Примечание**
>
> На данный момент поменять текст шаблона нельзя.

Если у вас есть активная кампания в Яндексе, то в сообщении будет указан идентификатор кампании.

![Атрибуция в WhatsApp](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/attribution-whatsapp.png)

Клиент может отредактировать сообщение — стереть текст шаблона и написать свое сообщение.

| Описание | Текст сообщения | Примечание |
| :--- | :--- | :--- |
| Сообщение для всех организаций | «Обращение из Яндекс Карт Здравствуйте! Меня заинтересовало ваше предложение» | |
| Сообщение для организации с активной кампанией в Директе | «Обращение №D5768768686 из Яндекс Карт Здравствуйте! Меня заинтересовало ваше предложение» | В номере обращения: D — обозначение Директа, 5768768686 — идентификатор баннера кампании в Директе |
| Сообщение для организации с активной кампанией в Яндекс Бизнесе | «Обращение №B47687686 из Яндекс Карт Здравствуйте! Меня заинтересовало ваше предложение» | В номере обращения: B — обозначение Яндекс Бизнеса, 47687686 — номер кампании в Яндекс Бизнесе |

## Статистика

Чтобы оценить эффективность продвижения организаций, отслеживайте статистику показов, кликов и достижения заданных целей.

Статистика по показам в Картах доступна в Мастере отчетов. Для анализа размещения постройте отчет с группировкой **Вид размещения** → **Карты**.

В отчете отображаются все ключевые метрики, а также данные по форматам: зеленая метка, поисковая выдача карт, карточка организации.

Чтобы оценить эффективность продвижения в карусели **Похожие организации**, добавьте группировки **Категория запроса** и **Упоминание брендов в запросах**.
