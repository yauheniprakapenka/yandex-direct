# Как запустить показы для товаров

## Требования к объявлениям c товарами для галереи

- По ссылке открывается страница с одним товаром или каталогом товаров.
- Актуальны наличие товара и его окончательная цена в объявлении.
- На странице есть активные кнопки «Купить» или «Добавить в корзину» и они соответствуют пользовательским сценариям. Дополнительные действия, после которых будет известна окончательная цена товара, такие как: «Заказать звонок», «Вызвать замерщика» не подходят для товарной галереи.
- Ссылка ведет на сайт, где можно приобрести продвигаемые товары. Переадресация на другие сайты не допускается.
- Рекомендуем добавлять фотографию товаров или каталога товаров на белом фоне без дополнительных символов.

## Создание кампании

Нажмите **Добавить** → **Кампанию**.

- В Мастере кампаний выберите **Товарная кампания** и создайте объявления для товаров и страниц каталога.
- В режиме эксперта выберите **Единая перфоманс-кампания** и создайте товарные объявления, страницы каталога или текстово-графические объявления.

> **Внимание**
>
> С 30 июня текстово-графические объявления в Единых перфоманс-кампаниях станут доступны только для редактирования. Для задач, которые раньше решались с помощью текстово-графических объявлений, можно будет создавать комбинаторные.

Товарные объявления и страницы каталога можно создать на основе YML-фида, сайта или CSV-файла.

## Как настроить показы товаров в товарной галерее

### Товарные объявления

**Единая перфоманс-кампания**

1. В настройках Единой перфоманс-кампании в блоке **Места показа** выберите **Поиск, Рекламная сеть и Карты** или **Ручная настройка** → **Поиск** → **Товарная галерея на поиске**.
2. Добавьте в группу товарное объявление.
3. Укажите источник для объявления: сайт, CSV-файл или YML-фид.

Заполните как можно больше полей в фиде. Для произвольного фида YML важно указывать параметры `url`, `price`, `picture`, `typePrefix`, `vendor`, `model`, для упрощенного фида YML — `url`, `price`, `picture`, `name`.

Важно передавать характеристики и параметры товаров через элемент `param`:

- Пол/Gender.
- Цвет/Colour или Color, можно передавать через запятую.
- Размер/Size.
- Материал/Material.

Укажите параметры для всех однотипных товаров, чтобы точнее подбирать объявления под конкретные запросы пользователей.

Один элемент `param` описывает только одну характеристику, при этом элемент `offer` может содержать любое количество `param`.

**Товарная кампания**

1. В настройках Товарной кампании укажите ссылку на ваш сайт. Директ автоматически предложит объявления для найденных на нем товаров.
2. Укажите источник для объявлений для товаров — сайт или фид по ссылке или с компьютера. Чтобы в объявлениях всегда были актуальные цены и информация о наличии товаров, используйте товарный фид.

После запуска кампании настроенные объявления для товаров начнут попадать в товарную галерею.

### Текстово-графические объявления

Чтобы текстово-графические объявления отбирались для показа в товарной галерее:

1. В настройках Единой перфоманс-кампании в блоке **Места показа** выберите **Поиск, Рекламная сеть и Карты** или **Ручная настройка** → **Поиск** → **Товарная галерея на поиске**.
2. Добавьте в группу текстово-графическое объявление.
3. Загрузите изображение товара в настройках объявления.

Подбирайте качественные изображения на белом фоне без текста, вотермарок и визуальных эффектов. На картинке должен быть четко виден товар.

4. Укажите цену товара.

Задайте цену без префикса. Если на товар есть скидка, укажите старую цену — скидка отобразится в карточке.

5. Проверьте, что добавленная ссылка ведет на страницу товара.

## Как настроить страницы каталога в товарной галерее

### Страницы каталога

**Единая перфоманс-кампания**

1. В настройках Единой перфоманс-кампании в блоке **Места показа** выберите **Поиск, Рекламная сеть и Карты** или **Ручная настройка** → **Поиск** → **Товарная галерея на поиске**.
2. Добавьте в группу объявление для страниц каталога.
3. Укажите источник для объявления: YML-фид, CSV-файл или сайт.
4. С помощью фильтров определите, какие каталоги хотите продвигать.

**Товарная кампания**

В Товарной кампании достаточно включить объявления для страниц каталога — и они смогут показываться в товарной галерее. В качестве источника для объявлений используйте сайт или CSV-файл.

### Текстово-графические объявления

1. В настройках Единой перфоманс-кампании в блоке **Места показа** выберите **Поиск, Рекламная сеть и Карты** или **Ручная настройка** → **Поиск** → **Товарная галерея на поиске**.
2. Добавьте в группу текстово-графическое объявление.
3. Загрузите изображение каталога в настройках объявления.
4. Проверьте, что добавленная ссылка ведет на каталог товаров.

## Чек-лист по настройке показов

### Настройка показов в Товарной галерее

С помощью Единой перфоманс-кампании

#### Место показа

- В настройках Единой перфоманс-кампании в блоке «Места показа» выбрали «Товарная галерея на поиске»

#### Структура кампании

- При необходимости провели сегментацию по категории, устройствам или региону таргетинга внутри одной кампании на уровне групп объявлений
- Создали товарные объявления и страницы каталога в одной рекламной кампании, чтобы ускорить обучение стратегии и получить больше конверсий
- Если для продвижения некоторых товаров необходимо установить разные целевые показатели, распределили их на отдельные группы в рамках одной кампании и установили на уровне группы корректировки цены конверсии или доли рекламных расходов. [Подробнее](#)

#### Стратегия

- Выбрали стратегию «Максимум конверсий»
- Задали стратегию с оптимизацией по ДРР, если используете электронную коммерцию
- Если купить товар можно разными способами, используйте все возможные варианты цели для оптимизации: покупка, добавление в корзину, звонок и другие
- Задали такую цену или ценность конверсии, которые соответствуют историческим данным и бизнес-показателям. [Инструкция](#)
- Рассчитали недельный бюджет таким образом, чтобы его хватило на покупку минимум 10 конверсий по самой дорогой из целей
- Задали в стратегии автоматическую модель атрибуции

#### Таргетинги

- Выбрали все категории запросов и упоминания брендов в запросах автотаргетинга
- Не используете название своего бренда в минус-фразах

#### Объявления

- Запустили товарные объявления и страницы каталога с генерацией по YML-фиду для всего ассортимента магазина
- Указали в фиде все обязательные элементы, описывающие товар: название, ссылку, изображение, цену
- Передаёте в фиде характеристики и параметры товаров через элемент param: пол, цвет, размер, материал. [Примеры фидов по категориям](#)
- Указали тег `<available>` для каждого офера, если хотите использовать фильтр по наличию товара. [Требования к YML-фиду](#)
- В фид добавили элемент `<collections>` и передаёте в нём информацию о каталогах: id, ссылку, изображение, название и описание. [Подробнее о каталогах товаров в фиде](#)
- Добавили в фид качественные изображения товаров и каталогов на белом фоне
- Убедились, что заглавные буквы не используются в фиде необоснованно
- Проверили, актуальна ли информация о наличии предложений и их цены

## Статистика

Статистика по показам в товарной галерее доступна в Мастере отчетов. Для анализа размещения постройте отчет с группировкой **Вид размещения** → **Товарная галерея**.

Чтобы проанализировать показы, клики, конверсии и другие показатели объявлений разных типов, используйте группировку **Тип объявления**.
