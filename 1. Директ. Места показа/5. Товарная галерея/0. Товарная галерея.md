# Товарная галерея

Товарная галерея — карусель с карточками товаров из разных магазинов или авто от разных продавцов, которая показывается в результатах поиска. Алгоритмы самостоятельно определяют оптимальное место показа галереи, чтобы увеличить вероятность конверсии. В результатах поиска может быть показано до двух товарных галерей с разными объявлениями. Показ двух товарных галерей недоступен в мобильной версии.

![Товарная галерея](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/product-gallery-2.png)

## Преимущества

- Показывает предложения, цены, акции и скидки в привлекательном и заметном формате.
- Помогает пользователю определиться с выбором.
- Повышает конверсионность продвижения товаров и автомобилей.
- Подходит бизнесу любого масштаба, в том числе небольшим магазинам.

## Особенности

В товарной галерее могут быть показаны:

- комбинаторные или текстово-графические объявления из Единой перфоманс-кампании о товарах — с изображением, ценой и ссылкой на товар;
- комбинаторные или текстово-графические объявления из Единой перфоманс-кампании о каталогах — с изображением и ссылкой на каталог;
- товарные объявления и страницы каталога из Единой перфоманс-кампании;
- объявления для товаров и страниц каталога из Товарной кампании.

> **Внимание**
>
> С 30 июня текстово-графические объявления в Единых перфоманс-кампаниях станут доступны только для редактирования. Для задач, которые раньше решались с помощью текстово-графических объявлений, можно будет создавать комбинаторные.

Страницы каталога в товарной галерее доступны только для товаров. Добавить в нее подборки автомобилей или предложений других тематик, в том числе услуг, не получится.

Объявления в товарной галерее на поиске показываются по ключевым фразам или автотаргетингу.

В товарную галерею может попасть более 5 карточек одного бизнеса, то есть в выдаче могут быть карточки только одного магазина. При этом один и тот же товар не повторяется — пользователю показываются разные товары, соответствующие его запросу (например, несколько моделей прямых диванов). В мобильной версии товарная галерея может состоять из одного товара.

Также в товарной галерее могут отображаться товары, похожие на основной запрос пользователя. Такие товары будут показаны с пометкой «Похожий товар». Чтобы ваши товары могли показаться в таком формате, выберите категории **Альтернативные запросы** и **Запросы с упоминанием брендов конкурентов** в настройках кампании в разделе **Автотаргетинг**.

### Характеристики товаров

В товарной галерее на объявлениях с товарами могут отобразиться характеристики, которые помогают пользователям сделать выбор при покупке. Объявления с такими характеристиками увеличивают количество конверсий в среднем на 4%.

Характеристикой товара могут быть разные параметры: диагональ экрана телевизора, цвет одежды, условия оплаты и другие. Для объявлений система выбирает характеристики, которые могут повысить вероятность клика и последующей конверсии на сайте.

Характеристики на объявлениях формируются из трех источников:

- Из фида с товарами. Характеристики нужно добавить с помощью параметра `property name`. В фид можно добавить до 10 характеристик.
- С продвигаемого сайта. Система проанализирует сайт и найдет ключевые характеристики товара и бизнеса, которые могут повлиять на решение пользователей о покупке.
- По аналогичным товарам из других магазинов. Чтобы расширить список характеристик товара и повысить вероятность конверсии, Директ проанализирует, какой товар вы продвигаете, найдет его в других магазинах и скопирует его характеристики.

Если найденные на сайте и в других источниках типы характеристик совпадают с теми, что вы указали в фиде, будет использоваться характеристика из фида. Это позволит управлять отображением наиболее значимых для вас параметров.

### Как выбираются объявления для показа

Выбор объявлений для показа происходит на основе VCG-аукциона. Он оценивает комбинацию ставки, коэффициента качества и прогноза CTR для конкретного показа. В галерею попадают объявления с лучшим сочетанием показателей.

## Места показов товарной галереи

Для всех товаров, кроме автомобилей:

- Поисковая выдача.
- Результаты поиска в картинках.
- Выдача в поиске по товарам.
- В приложении Едадил в результатах поиска.

Показы автомобилей будут только в поисковой выдаче.
