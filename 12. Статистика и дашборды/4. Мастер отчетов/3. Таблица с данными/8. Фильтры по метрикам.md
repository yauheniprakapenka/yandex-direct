# Фильтры по метрикам

Помимо фильтров, которые есть в настройках отчета и которые позволяют отсеивать данные на уровне группировок, в Мастере отчетов доступны отдельные фильтры для столбцов с метриками.

![Фильтры по метрикам](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filters-by-metrics-1.png)

Например, можно отфильтровать данные по определенному CTR или оставить только те кампании, где были клики.

В настройках фильтра доступно несколько условий соответствия на выбор.

![Условия фильтра](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filters-by-metrics-2.png)

При выставлении условий фильтра данные в отчете отобразятся в режиме реального времени.

![Фильтр в отчете](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filters-by-metrics-3.png)

Если нужно изменить условия фильтра, кликните по его значению в заголовке столбца.
Чтобы убрать фильтр, нажмите крестик рядом с условием в столбце или в настройках столбца выберите **Удалить фильтр**.

![Удалить фильтр](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filters-by-metrics-4.png)
