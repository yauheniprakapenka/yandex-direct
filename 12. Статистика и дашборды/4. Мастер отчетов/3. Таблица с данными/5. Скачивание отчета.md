# Скачивание отчета

Отчеты можно скачать в формате XLXS и CSV. Отметьте в настройках формат для скачивания файла, который будет использоваться по умолчанию.

![Скачивание отчета](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-download.png)

Если выбран один из вариантов — XLSX или CSV, то при нажатии на кнопку **Скачать** загрузка файла с отчетом начнется сразу. Если указаны оба формата, при нажатии на кнопку **Скачать** можно будет выбрать формат файла.

![Выбор формата](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-format.png)
