# Таблица с данными

Это отчет по группировкам и метрикам, основному периоду или сравниваемым, целям, атрибуции и фильтрам, которые вы указали в настройках.

Группировки по названиям и идентификаторам кампаний, групп и объявлений кликабельные. При клике по ним вы попадете на страницу редактирования соответствующего элемента.

## Подразделы

- **Сворачивание и разворачивание экрана** — управление видимостью секций таблицы.
- **Древовидные таблицы** — отображение данных в виде иерархической структуры.
- **Плотность отображения данных** — настройка компактности таблицы.
- **Детализация** — углубленный анализ данных по отдельным элементам.
- **Скачивание отчета** — экспорт данных в файл.
- **Настройки столбцов** — добавление, удаление и изменение порядка столбцов.
- **Режим массовых действий** — выполнение операций сразу над несколькими строками.
- **Фильтры по метрикам** — ограничение данных по значениям метрик.
- **Сортировка** — упорядочивание данных по столбцам.
- **Закрепление колонки** — фиксация столбца при горизонтальной прокрутке.
- **Передвижение столбцов** — изменение порядка столбцов перетаскиванием.
- **Цветовое форматирование** — условная окраска ячеек для наглядности.
- **Дополнительные опции** — прочие настройки отображения данных.
