# Фильтры

Фильтры позволяют отсеивать ненужные данные из отчета и фокусироваться на наиболее важных. Показатели, для которых доступны фильтры, собраны в группы.

По умолчанию в настройках установлен фильтр, который исключает архивные кампании, если аккаунту больше полугода. Это ускоряет создание отчетов в больших аккаунтах.

> **Важно**
>
> В настройках отчета доступны только фильтры по группировкам. Фильтры по метрикам можно использовать для [настройки столбцов](ru/statistics/new-reports-wizard/data-table/filters-by-metrics).

![Фильтры](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter.png)

Чтобы увидеть статистику по конкретной кампании, используйте фильтр по названию или номеру кампании. А если нужна статистика только по кампаниям с размещением в Рекламной сети Яндекса, выберите **Тип площадки** → **Сети**.

![Фильтр по кампании](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter-campaign.png)

### Как работают сочетания фильтров

- Фильтры работают по принципу пересечения блоков между собой по логическому оператору «И».
- Если внутри фильтров выбрано несколько вариантов:
  - для операторов «Равно» и «Содержит» применяется логический оператор «ИЛИ»;
  - для операторов «Не равно» и «Не содержит» — логический оператор «И».
- Фильтры по текстовым данным, например по названию кампании, можно использовать несколько раз, чтобы составить сложные условия. Текстовый фильтр с оператором «Содержит» также можно выбрать несколько раз.

### Пример

| Условие для фильтра | Названия кампаний | Результат фильтрации |
| :--- | :--- | :--- |
| «содержит бренд ИЛИ брэнд» И «содержит Москва ИЛИ мск» И «не содержит промо И акция» | РФ_Бренд_Промо_весна_2025, РФ_Бренд_Основная_весна_2025, МСК_Бренд_Промо_весна_2025, МСК_Бренд_Основная_весна_2025, МСК_Целевые_Основная_весна_2025, рф_брэнд_промо_лето_2025, рф_брэнд_основная_лето_2025, мск_брэнд_промо_лето_2025, мск_брэнд_категории_лето_2025, мск_целевые_основная_лето_2025, РФ_бренд_акция_осень_2025, РФ_бренд_основная_осень_2025, москва_бренд_акция_осень_2025, москва_бренд_основная_осень_2025 | МСК_Бренд_Основная_весна_2025, мск_брэнд_категории_лето_2025, москва_бренд_основная_осень_2025 |

Если тип фильтра предполагает конечный набор значений, их можно выбрать из списка, кликнув по названию фильтра.

![Выбор из списка](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter-list.png)

Кроме того, фильтры поддерживают вставку списка значений — например, можно указать ID ваших кампаний через запятую или перенос строки и добавить в фильтр только их по кнопке **Выбрать все**.

![Выбрать все](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter-select-all.png)

Также доступны строковые фильтры с условиями — например, можно выделить кампании, в названиях которых есть слова «бренд» и «Москва» и нет слова «промо».

![Строковые фильтры](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter-conditions.png)

- Для фильтров с оператором «содержит» можно использовать специальные символы:

  `^` — соответствует одному любому символу. Например, под условие фильтра «Название кампании содержит бр^нд» попадут кампании с названиями «Поиск_бренд_Мск» и «РСЯ_брэнд_РФ».

  `*` — соответствует любому количеству любых символов. Например, под условие фильтра «Название кампании содержит Промо*Поиск» попадут кампании с названиями, «Директ_Промо_МСК_Поиск», «Промо_РФ_Поиск_2».

Если набор фильтров получился большой, вы можете свернуть настройки через кнопку **Фильтры**.

Удалить все фильтры поможет кнопка **Сбросить все**.

![Сбросить все](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-filter-drop-all.png)
