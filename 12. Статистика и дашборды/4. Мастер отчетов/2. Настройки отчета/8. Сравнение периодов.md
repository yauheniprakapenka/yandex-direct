# Сравнение периодов

В отчетах можно посмотреть данные по двум периодам одновременно. Для этого нажмите **Сравнение** и выберите один из предлагаемых периодов или укажите свой. Эти настройки учитываются и при [сохранении отчетов](ru/statistics/new-reports-wizard/settings/save-report): например, можно создать отчет со сравнением «неделя к неделе», сохранить и открывать из раздела **Мои отчеты**.

![Сравнение периодов](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-compare.png)

По умолчанию для сравнения доступны:

- Предыдущий равный период;
- Предыдущая неделя;
- Предыдущий месяц;
- Предыдущий год.

При настройке периодов сравнения можно выбрать произвольные интервалы и не ограничивать пересечения дат и порядок следования. При необходимости можно в один клик поменять местами ![Поменять местами](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/change.png) периоды сравнения.

> Если вы работаете с большими объемами данных, закрепите колонку, чтобы нужная статистика отображалась в отчете при горизонтальной прокрутке.
