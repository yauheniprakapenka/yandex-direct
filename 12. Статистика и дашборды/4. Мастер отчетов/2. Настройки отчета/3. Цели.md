# Цели

В этом блоке вы можете выбрать цели, по которым будут рассчитываться конверсионные метрики, например конверсии, CPA и другие. Цели сгруппированы по счетчикам Метрики, которые используются в ваших кампаниях. Можно свернуть или развернуть список для каждого счетчика.

А также воспользоваться поиском по названиям или номерам целей и счетчиков.

![Выбор целей](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-goal.png)

Если не выбрано ни одной цели, по конверсионным метрикам будет отображаться количество целевых визитов, в которых была достигнута хотя бы одна из целей.

Если указано несколько целей, настройте отображение статистики по конверсионным метрикам. Для этого выберите **Суммарно по всем выбранным целям**, **Отдельно по каждой цели** либо оба варианта.

![Настройки отображения](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-display-settings.png)

Также вы можете задать отдельные настройки для каждой метрики. Например, смотреть количество конверсий по всем целям, а CR — по каждой отдельно.

Чтобы колонки отображались по каждой цели отдельно, выберите до 20 целей. Если целей больше 20, отобразить можно только общее количество конверсий по всем отмеченным целям.

![Конверсии по целям](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-conversion.png)

Если в **Настройках отображения** целей выбран вариант показа конверсии **Отдельно по каждой цели**, вы можете настроить порядок столбцов с целями в отчете. Для этого перейдите на вкладку **Цели** и переместите цели на нужные позиции в списке.

![Порядок целей](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/goal-move.png)

Чтобы отменить выбор всех целей, воспользуйтесь кнопкой **Удалить все**.
