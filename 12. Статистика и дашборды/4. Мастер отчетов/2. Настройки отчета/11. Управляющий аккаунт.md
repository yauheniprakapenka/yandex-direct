# Управляющий аккаунт

Если вы работаете в Директе под управляющим аккаунтом, то можете просматривать статистику сразу по нескольким аккаунтам или выбрать один.

![Выбор аккаунтов](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-account-2.png)

Также для управляющего аккаунта доступна группировка «Аккаунты». Ее можно добавить в отчеты, чтобы увидеть общие данные по разным аккаунтам, например по расходам в них.

![Группировка Аккаунты](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-account-group.png)

![Группировка Аккаунты в отчете](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/stat-account-group-2.png)

Чтобы добавить в отчет цели из доступных аккаунтов, выберите нужные аккаунты в настройках отчета. В окне настройки целей выберите аккаунт, из которого хотите добавить цели и отметьте нужные.

![Цели из доступных аккаунтов](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/goal-account.png)
