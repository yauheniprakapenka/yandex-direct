# Отчет «Поисковые запросы»

Отчет поможет проанализировать запросы, по которым были показаны ваши объявления. Найти его можно в **Библиотеке отчетов** → раздел **Оптимизация размещения**.

При переходе из Библиотеки откроется отчет по всем кампаниям в Поиске.

![Библиотека отчетов](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/index-search.png)

В отчет по конкретной кампании можно перейти со страницы списка кампаний. Для этого справа от названия кампании нажмите ![Меню](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/kebab.png) и выберите пункт **Поисковые запросы**. Этот пункт доступен только для кампаний с размещением в Поиске.

![Пункт «Поисковые запросы»](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/search-queries.png)

После этого откроется отчет с фильтром по выбранной кампании.

![Отчет с фильтром по кампании](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/filter-search.png)

Вы также можете перейти в отчет из любого другого отчета с помощью навигационной панели.

![Навигационная панель](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/navigation-bar.png)

Если опция **Наследовать фильтры** включена, то когда вы перейдете из навигации в отчет, сохранятся фильтры, которые используете в текущем отчете.

## Настройки отчета

По умолчанию в отчет добавлены основные группировки, метрики, фильтры и используются цели и атрибуция из настроек кампаний.

![Настройки отчета](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/settings-search.png)

При необходимости вы можете скорректировать настройки и добавить нужные [группировки и метрики](ru/statistics/new-reports-wizard/settings/group-metrics). Например, интернет-магазинам важно анализировать результаты не только по количеству конверсий и CPA, но и в разрезе дохода, ДРР, ROI и прибыли. Эти метрики можно добавить в отчет.

Если вы хотите, чтобы в отчете по поисковым запросам всегда открывались добавленные настройки, сохраните их. Для этого в правом верхнем углу нажмите ![Сохранить](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/rw-save.png) → **Сохранить настройки**.

![Сохранить настройки](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/save.png)

### Группировки

В отчет по умолчанию добавлены группировки, которые помогают анализировать поисковые запросы:

- **Поисковый запрос** — запрос, который ввел пользователь.
- **Тип условия показа** — параметр в настройке кампании, который инициировал показ объявления: ключевая фраза, автотаргетинг, условие подбора аудитории, офферный ретаргетинг.
- **Ключевая фраза** — фраза, которая инициировала показ объявления.
- **Тип соответствия** — связь между исходной ключевой фразой и фразой, по которой показано объявление.
- **Категория запроса** — категория поискового запроса, по которому было показано объявление.
- **Упоминание брендов в запросах** — упоминание брендов в поисковых запросах, по которым было показано объявление.

Данные доступны только за последние 180 дней.

В строке **Другие поисковые запросы** представлены следующие данные:

- Еще не обработанные запросы — запросы, которые были сделаны менее часа назад (даже если по ним были клики).
- Запросы, по которым не было кликов и было незначительное количество показов (за исключением запросов, по которым клики уже были ранее или в рамках других кампаний — они указаны в отдельных строках).

#### Тип условия показа

| **Тип условия показа** | **Что означает** | **Пример** |
| :--- | :--- | :--- |
| Фраза | Объявление показано по запросам, в которых ключевая фраза содержится целиком. [Подробнее о ключевых фразах](ru/keywords/keywords) | Объявление с ключевой фразой _квест на день рождения_ показано по запросу _проведение квеста на день рождения_ |
| Автотаргетинг | Объявление показано всем, кто ищет ваши товары или услуги, на основе содержания объявления, без использования ключевых фраз. В режиме реального времени система сопоставила заголовок, текст и страницу перехода в объявлении с поисковыми запросами пользователей. Яндекс может проверить подбор по автотаргетингу на корректность только за последние 60 дней. [Подробнее об автотаргетинге](ru/impression-criteria/autotargeting) | Объявление с текстом про путешествия в космос и страницей перехода `в-космос.рф` показано по запросу _туры на марс_. |
| Условие подбора аудитории | Представляет собой комбинацию целей/сегментов Яндекс Метрики и сегментов Яндекс Аудиторий. Объявление показано только тем пользователям, которые вас интересуют. [Подробнее о подборе аудитории](ru/impression-criteria/retargeting-lists) | По ключевой фразе _купить диван_ и условию ретаргетинга _посетил сайт_ объявление покажется пользователю, который напечатал в поиске «купить диван» и уже посещал сайт. |
| Интересы и соцдем | Объявление показано пользователям, чье поведение в интернете принадлежит к определенной группе. Краткосрочные интересы и привычки определяются по посещенным сайтам и приложениям, тематикам поисковых запросов, маршрутам из Яндекс Карт. [Подробнее о показах по интересам и привычкам](ru/products-text-image-ads/user-profile) | Объявление о распродаже авиабилетов или летней одежды будет показано пользователям, которые собираются в отпуск. |
| Офферный ретаргетинг | Объявление показано пользователям, которые уже взаимодействовали с товарными объявлениями или каталогами на вашем сайте. | Пользователь посмотрел товар или каталог на сайте. Он увидит эту информацию в объявлении в РСЯ. |

#### Категория запроса

| **Категория запроса** | **Что означает** | **Пример** |
| :--- | :--- | :--- |
| Целевые запросы | Объявление точно отвечает на запрос пользователя. | Для пользователя «Мир рюкзаков» с объявлением «Рюкзак для ноутбука» целевой запрос — _купить рюкзак для ноутбука_ |
| Альтернативные запросы | Пользователь ищет продукт, который можно заменить продвигаемым. При этом объявление также может удовлетворить запрос. | Объявление «Рюкзак для ноутбука» показано по запросу _купить сумку для ноутбука_ |
| Сопутствующие запросы | Запросы по продуктам, которые могут быть интересны вместе с продвигаемым товаром или услугой. | Объявление «Рюкзак для ноутбука» показано по запросу _купить ноутбук_ |
| Запросы с упоминанием конкурентов | Поиск продвигаемого продукта у конкурентов. | Объявление «Рюкзак для ноутбука» показано по запросу _купить рюкзак для ноутбука Планета рюкзаков_ |
| Широкие запросы | Запрос шире, чем предложение. | Объявление «Рюкзак для ноутбука» показано по запросу _купить рюкзак_ |
| Не определена | Запрос не подходит ни под одну из категорий. | |

#### Тип соответствия

| **Тип соответствия** | **Что означает** | **Пример** |
| :--- | :--- | :--- |
| Пословное | Объявление показано по запросам, целиком содержащим заданные ключевые фразы. | Объявление с ключевой фразой _квест на день рождения_ показано по запросу _заказать квест на день рождения_. |
| Семантическое | Объявление показано по запросам, которые семантически уточняют фразу: содержат опечатки, синонимы, более конкретные слова, переформулировки. Яндекс может проверить подбор по семантическому соответствию на корректность только за последние 60 дней. | Объявление с ключевой фразой _приобрести квест для детей_ показано по запросу _квест для детей купить_, а с ключевой фразой _где поесть_ — по запросу _рестораны быстрого питания_. |

## Как работать с отчетом

Выберите в настройках период, за который хотите посмотреть данные. Добавьте группировки и метрики, если тех, что уже есть в отчете, недостаточно. После этого укажите, по каким целям будете анализировать размещение, и при необходимости измените модель атрибуции.

Данные в отчетах можно отфильтровать по группировкам и метрикам.

Фильтры по группировкам находятся в настройках отчета. [Подробнее о фильтрах](ru/statistics/new-reports-wizard/settings/filter)

![Добавление фильтров](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/add-filters-s.png)

Чтобы отфильтровать данные по метрикам, например оставить только информацию по поисковым запросам, по которым было более 10 конверсий, воспользуйтесь фильтром в столбце. В названии столбца нажмите ![Меню](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/kebab.png) → **Добавить фильтр** и укажите условие.

![Добавить фильтр](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/kebab-filtr.png)

![Условие фильтра](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/condition.png)

Вы также можете анализировать эффективность поисковых запросов и ключевых фраз в разрезе задач. Например, насколько эффективна та или иная фраза или запрос с точки зрения результатов для бизнеса и KPI.

### Добавление минус-фраз из отчета

Помимо аналитики поисковых запросов, прямо из отчета можно добавить нерелевантные запросы в минус-фразы на кампанию и группу объявлений.

Выберите нужные фразы и нажмите **Добавить в минус-фразы**.

![Выбор фраз](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/check.png)

В открывшемся окне появится список выбранных фраз.

![Список фраз](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/library/phrases.png)

Вы можете отредактировать выбранную фразу — например, добавить операторы `+`, `!`, `[]`. По умолчанию добавляемая фраза указана в кавычках `""`. Вы также можете выбрать уровень, где будет работать минус-фраза, — группа объявлений или кампания. Нажмите кнопку **Сохранить**.

Если при добавлении выбранных запросов в минус-фразы вы превысите лимиты на количество символов на группу или кампанию, будет выведено предупреждение.

Если новая минус-фраза совпадает с ключевой фразой, вы также увидите предупреждение.

Расширить список минус-фраз можно и в Мастере отчетов с помощью группировки **Поисковый запрос**.

[Как добавить минус-фразы из отчета](ru/statistics/new-reports-wizard/data-table/mass-action#negative-keywords)

### Добавление ключевых фраз из отчета

Отметьте запросы, которые хотите добавить в список фраз и нажмите **Добавить в ключевые фразы**.

![Добавить в ключевые фразы](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/add.png)

В открывшемся окне можно отредактировать фразу, добавить операторы или указать ставку, если в кампании выбрана стратегия **Максимум кликов с ручными ставками**.

![Редактирование фразы](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/add-key.png)

Нажмите **Сохранить**.

Уникальные фразы из выбранных запросов будут добавлены в ключевые фразы указанной группы объявлений. Если новые фразы не поместятся в одну группу, Директ автоматически сделает ее копию и добавит оставшиеся фразы туда.

Если добавить поисковые запросы в ключевые фразы не получается из-за превышения лимита группы объявлений на кампанию, вы можете уменьшить число добавляемых фраз, уменьшить количество фраз в исходных группах или количество групп в кампании.

Добавить запросы можно и в Мастере отчетов с помощью группировки **Поисковый запрос**.

[Как добавить запросы из отчета](ru/statistics/new-reports-wizard/data-table/mass-action#key-phrases)
