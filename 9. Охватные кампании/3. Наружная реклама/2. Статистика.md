# Статистика кампании

Статистика по вашему размещению будет доступна в Мастере отчетов.

Чтобы оценить работу кампании, добавьте метрики для наружной рекламы:

1. Нажмите **Группировки и метрики**.
2. Выберите **Метрики** → **Охватная реклама** → **Наружная реклама (DOOH)**.
3. Нажмите **Применить**.

Оценить статистику для наружной рекламы можно с помощью группировок **Тип рекламной конструкции**, **Партнер**, **Адрес экрана**, **Категория партнера**, **Номер экрана и сторона**. Чтобы добавить в отчет группировку:

1. Нажмите **Группировки и метрики**.
2. Выберите **Группировки** → **Наружная реклама**.
3. Выберите нужную группировку.
4. Нажмите **Применить**.

Для удобства вы можете сохранить отчет.
