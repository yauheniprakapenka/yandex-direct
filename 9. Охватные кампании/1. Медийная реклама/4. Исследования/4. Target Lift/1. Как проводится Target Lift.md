# Как проводится Target Lift

В основе исследования — анализ динамики целевых действий на сайте и в мобильном приложении. Для этого оценивается реальная аудитория кампании.

Из целевой аудитории кампании отбирается тестовая (90%) и контрольная (10%) группа. Тестовая группа включает пользователей, которые видели рекламу. Контрольная группа состоит из пользователей той же целевой аудитории, которым реклама намеренно не демонстрируется. Для создания контрольной группы используется [виртуальный аукцион](ru/research#virtual-auction).
