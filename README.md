# Яндекс Директ — оффлайн-справка

Локальная копия [справки Яндекс Директа](https://yandex.ru/support/direct/ru/) в формате Markdown. Удобно искать по репозиторию, работать без интернета и подключать AI-ассистента в Cursor.

Онлайн-источник статей: `https://yandex.ru/support/direct/ru/{path}` (для сверки и обновлений).

## Структура

Разделы пронумерованы и повторяют оглавление справки:

| № | Раздел |
| --- | --- |
| 1 | Места показа |
| 2–4 | Простой старт, Мастер кампаний, Товарная кампания |
| 5 | Продвижение приложений |
| 6 | Единая перфоманс-кампания |
| 7 | Реклама в мессенджерах |
| 9 | Охватные кампании |
| 11–17 | Объявления, статистика, фиды, управление, оплата, модерация, технологии |
| 18–21 | Обратная связь, Директ Про, агентствам, прочее |

Внутренние ссылки в файлах — `ru/...` (относительные пути справки). Изображения — с CDN Diplodoc.

## Работа с AI

В [AGENTS.md](AGENTS.md) — инструкции для агента: как отвечать по справке, разбирать отчёты и давать рекомендации.

**Типичный сценарий:** прикрепить выгрузку или скрин из Директа (Мастер отчётов, «Поисковые запросы», «Площадки») и указать:

- тип кампании и цель (лиды, продажи, охват);
- период;
- целевой CPA / ДРР, если есть.

Агент опирается на документацию из репозитория и отвечает структурой: вывод → что хорошо → проблемы → рекомендации.
