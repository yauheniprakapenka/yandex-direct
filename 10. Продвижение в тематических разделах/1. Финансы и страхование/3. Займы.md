# Займы β

Тематические разделы поиска позволяют пользователям рассчитать условия получения займа.

Продвижение в этих разделах доступно в режиме бета-тестирования для микрокредитных компаний (МКК) и микрофинансовых организаций (МФО), имеющих лицензию ЦБ РФ и включенных в государственный реестр.

Технические требования предусматривают возможность интеграции в соответствии с условиями, размещенными в разделе Займы — API для партнеров.

При ранжировании продвигаемых в тематическом разделе предложений учитываются следующие показатели:

- ставка (определяется заданной ценой целевого действия);
- показатель кликабельности;
- коэффициент релевантности.

## Как пользователь увидит персональные предложения

Объявления могут показываться на всех типах устройств в поиске по Финансам. После того как пользователь введет поисковый запрос или перейдет в раздел «Займы» в поиске по Финансам, в результатах выдачи появится тематический раздел для расчета условий получения займа.

В разделе для расчета условий получения займа пользователь делает следующие шаги:

1. Указывает сумму, срок и другие условия получения займа и нажимает кнопку **Рассчитать условия**.
2. Авторизуется в Яндекс ID или подтверждает профиль.
3. Получает список организаций, в которых есть подходящие предложения и отмечает интересующие. Все выбранные пользователем организации получат его заявку и могут рассчитать для него персональные условия выдачи займа.
4. Переходит к заполнению анкеты и знакомится с общими правилами оформления займов.
5. На основе данных, указанных в анкете, получает персональные предложения от интересующих его компаний. Если ни одно из предложений не подойдет, ему будут показаны объявления с похожими условиями. Пользователь также может изменить список выбранных компаний, чтобы увидеть больше предложений.
6. Переходит из объявления к оформлению выбранного займа на сайте компании.

## Как настроить

1. Настройте API-интеграцию с вашим сервисом. Это позволит:
   - Автоматически формировать и показывать предложения с учетом данных пользователя и характеристик его профиля.
   - Направлять пользователя на страницу оформления подобранного для него займа прямо из объявления.
   - Получать подробную статистику о результатах продвижения.
2. Настройте и запустите кампанию в Директе.

## Создание и настройка кампании в Директе

1. Нажмите **Добавить** → **Кампанию**.
2. На вкладке **Режим эксперта** выберите **Продвижение в тематических разделах** → **Финансы и страхование**.
3. В блоке **Продвигаемый продукт** выберите **Займы β**.
4. В блоке **Бюджет и цена целевого действия** задайте следующие параметры:
   - **Бюджет** — ограничение расхода в неделю.
   - **Цена целевого действия** — плата за каждый оформленный займ. Цена не должна быть ниже установленного минимального порога, иначе показ объявления прекратится. Подсказка о минимальном пороге отображается в интерфейсе, если указанная цена ниже его значения. Об изменении минимального порога менеджер сообщит заранее.
5. В блоке **География показов** выберите подходящие регионы — объявления будут показаны только той аудитории, которая сможет воспользоваться вашим предложением, так как находится в выбранном регионе показа или интересуется им. Вы можете:
   - **Перечислить страны, области, города** — в свободной форме начните вводить нужный регион, например, Тула или Москва. Или воспользуйтесь деревом регионов.
   - **Указать регион на карте** — если вас интересуют клиенты в узкой области.
6. При необходимости в блоке **Корректировки по географии** задайте коэффициенты корректировок для разных регионов показа.
7. В блоке **Интеграция** заполните параметры для API-интеграции:
   - **API** — укажите ссылку на API;
   - **Client ID** — ID пользователя для авторизации;
   - **Client Secret** — ключ для авторизации.
8. Запустите кампанию.

После запуска данные интеграции уйдут на проверку. Как только проверка будет закончена, статус кампании изменится на «Активна».

## Оплата продвижения

Оплата только за целевое действие — оформление займа.

Цена целевого действия списывается фиксированная, с учетом установленного недельного бюджета.

## Статистика

> **Важно**
> Чтобы оценить результаты продвижения, после оформления пользователем займа нужно передать данные о покупке в Директ через API. Рекомендуем передавать данные в течение 3 дней, максимальный срок — до 11 дней. После этого ключевые метрики отобразятся в Мастере отчетов и на странице **Кампании**.
> Если данные о конверсиях не будут переданы в Директ, статистика будет недоступна.

В Мастере отчетов для анализа используйте группировку **Тип кампании** → **Продвижение в тематических разделах** или **Вид размещения** → **Тематический раздел**. В отчете отображаются все ключевые метрики, в том числе данные о показах и конверсиях. В столбце **Конверсии** отображается количество оформленных займов.
