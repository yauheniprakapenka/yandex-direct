# Частые ошибки при подготовке фидов

| Ошибка | Описание |
|--------|----------|
| **Использование фида Яндекс Недвижимости** | В качестве фида с офферами нельзя использовать фид Яндекс Недвижимости. Также нельзя использовать корневой элемент из фида для Яндекс Недвижимости. Это приведет к сбоям при выгрузке. |
| **Ошибки оформления блока `location`** | Элементы `country`, `region`, `district`, `sub-locality-name` считаются устаревшими и не поддерживаются. Элементы `latitude` и `longitude` не могут быть пустыми. Если вы не хотите передавать координаты, не используйте эти элементы в фиде. |
| **Размещение элемента `url` внутри блока `sales-agent`** | В фиде с объявлениями элемент `url` поддерживается только вне блоков. В блоке `sales-agent` он не поддерживается. |
| **Использование элемента `creation-date`** | В фиде с объявлениями элемент `creation-date` не поддерживается. |
| **Некорректная стоимость звонка** | В фиде со ставками в элементе `cost-per-call` указывается стоимость звонка в копейках, а не в рублях. Значение может быть принято системой, но будет пересчитано до минимальной ставки. Из-за этого объявление может проиграть аукцион. |
| **Отсутствие в фиде идентификаторов ЖК и корпуса ЖК** | `yandex-building-id` и `yandex-house-id` — обязательные элементы в фиде. Их значения можно посмотреть в файле: `yandex-building-id` указан в третьем столбце, `yandex-house-id` — в седьмом. |

### Примеры ошибок

**Использование фида Яндекс Недвижимости (неправильно):**

Фид со ставками:
```
<campaign-feed xmlns="http://webmaster.yandex.ru/schemas/feed/realty/2010-06">
```

Фид с объявлениями:
```
<realty-feed xmlns="http://webmaster.yandex.ru/schemas/feed/realty/2010-06">
```

**Правильно:**

Фид со ставками:
```
<campaign-feed xmlns="http://webmaster.yandex.ru/schemas/feed/metarealty/campaigns/2024-12">
```

Фид с объявлениями:
```
<realty-feed xmlns="http://webmaster.yandex.ru/schemas/feed/metarealty/2024-12">
```

**Ошибки оформления блока `location` (неправильно):**

```
<location>
    <country>Россия</country>
    <district>Рязанская область</district>
    <address>Борки микрорайон</address>
    <latitude></latitude>
    <latitude></longitude>
    <apartment>1</apartment>
</location>
```

**Размещение `url` внутри `sales-agent` (неправильно):**

```
<sales-agent>
    <url>https://www.green-garden.ru</url>
    <category>developer</category>
</sales-agent>
```

**Использование `creation-date` (неправильно):**

```
<creation-date>2026-01-20T21:07:33+03:00</creation-date>
```
