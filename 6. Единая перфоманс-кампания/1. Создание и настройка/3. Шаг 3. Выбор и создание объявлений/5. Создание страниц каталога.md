# Создание страниц каталога

- [Модерация объявлений и запуск кампании](#модерация-объявлений-и-запуск-кампании)
- [Вопросы и ответы](#вопросы-и-ответы)

В группу можно добавить **только одно** активное объявление для страниц каталога. Объявления для страниц каталога можно настроить на основе:

- Сайта — только для товаров.
- Фида — для товаров (YML-фид) и тематик «Авто» и «Недвижимость» (XML-фид).
- CSV-файла — для всех тематик, которые можно продвигать в Директе.

1. Укажите источник для создания объявлений:

    **Страницы каталога, найденные на сайте**

    Робот Яндекса обходит страницы каталога на сайте. Чтобы робот точно определил страницы каталога, они должны быть размечены специальной строгой микроразметкой.

    Объявления для страниц каталога, найденных на сайте, можно сгенерировать только для товаров.

    **Фид по ссылке или с компьютера**

    - **CSV-файл** — можно использовать для генерации объявлений для страниц каталога с любыми товарами и услугами, которые можно продвигать в Директе. В одну группу можно загрузить только один файл.

        Требования к файлу:

        - Формат CSV с кодировкой UTF-8.
        - Запятая «,» или точка с запятой «;» в качестве разделителя.
        - В шапке файла должен быть перечислен весь набор столбцов: `Url`, `Title`, `Description`, `Offer minimal price`, `Currency`, `Image url 1`–`Image url 5`.
        - Ограничение по количеству строк — 1000.

        Описание полей:

        | Название | Описание |
        |---|---|
        | `Url` | Ссылка на страницу каталога. Каждая ссылка уникальна. Обязательное поле. |
        | `Title` | Название каталога. До 56 символов. Обязательное поле. |
        | `Description` | Описание. До 81 символа, до 15 знаков препинания. Рекомендуем указать. |
        | `Offer minimal price` | Минимальная цена товара в категории. Целое число без разделителя. |
        | `Currency` | Буквенный код валюты (RUB, USD, EUR). Обязательное поле, если заполнено поле Offer minimal price. |
        | `Image url 1`–`5` | Ссылки на изображение. Рекомендуем указать. |

        [Скачать пример файла](https://yastatic.net/s3/doc-binary/src/support/direct/ru/catalogs_example.csv)

        Возможные ошибки при загрузке файла:

        - Неверная кодировка — проверьте, что файл в кодировке UTF-8.
        - Файл не может быть пустым — проверьте, что файл содержит строки с каталогами.
        - Список не может быть больше 1000 строк.
        - Поле не должно быть пустым — проверьте, что все обязательные поля заполнены.
        - Заголовок должен включать только поддерживаемые символы.
        - Длина слова не может быть больше — заголовок не более 56 символов, текст — не более 81.
        - Незакрепленная запятая — проверьте расстановку запятых и кавычек.
        - Нельзя указывать одинаковые значения — все ссылки в поле URL должны быть уникальными.

    - **YML или XML-фид** — загрузите фид в формате YML (для товаров) или XML (для автомобилей и недвижимости). Используйте фид со специальным тегом `<collections>` для передачи информации о каталогах. Подробнее о каталогах [товаров](ru/feeds/requirements-yml#collections), [автомобилей](ru/feeds/requirements-other-categories#autos), [объектов недвижимости](ru/feeds/requirements-other-categories#real-estate).

    Пользователи 1С-Битрикс могут быстро сформировать YML-фид со страницами каталога с помощью [инструкции](https://marketplace.1c-bitrix.ru/blog/yandex-market-collection/).

    Если вы работаете с платформой InSales и настроили автоматическое обновление фида, то в него добавится информация о каталогах товаров — дополнительных действий не требуется.

2. В блоке **Фильтр** выберите, какие каталоги хотите продвигать. Условия объединяются оператором «И».

    Если объявления генерируются по сайту, доступны два фильтра:
    - **Страницы каталога, соответствующие товарам** — Директ создаст объявления только для каталогов с товарами, которые уже продвигаются в кампании.
    - **Конверсионные страницы** — Директ создаст объявления только для каталогов с товарами, по которым уже были конверсии по целям кампании.

3. Для объявлений из YML/XML-фида в блоке **Тексты** настройте заголовки и тексты:
    - Можно использовать автоматические заголовки и тексты или ручные (заголовок из `<name>`, текст из `<description>`).
    - Задайте текст по умолчанию — будет использоваться, если не получится сгенерировать текст автоматически.

При необходимости фид можно заменить на этапе создания или редактирования объявления. Директ постарается переиспользовать фильтры. Если в фиде нет элементов, отвечающих условию фильтра, то он будет сброшен.

## Модерация объявлений и запуск кампании

После создания ваше объявление будет проверено на соответствие [установленным требованиям](ru/moderation/adv-rules). Нажмите кнопку **Отправить на модерацию** и подтвердите свое согласие с условиями [оферты](https://yandex.ru/legal/oferta_direct/). Обычно модераторы проверяют объявление в течение нескольких часов. В выходные и праздничные дни модерация может занять больше времени.

Результаты проверки вы получите письмом на адрес, указанный при создании кампании.

Показы объявлений начнутся только после того, как вы [оплатите продвижение](ru/payments/payment-methods).

## Вопросы и ответы

### Не формируются объявления для страниц каталога

Внимание

Убедитесь, что на счете есть деньги. Генерация объявлений не начнется, если нет средств.

**Не формируются объявления по фиду (по ссылке или с компьютера)**

- Объявления могли не сформироваться, если не были соблюдены [требования к фиду](ru/feeds/requirements) или требования к CSV-файлу.
- Если использовались отдельные каталоги, отобранные с помощью фильтров, проверьте условия:
    - **Каталог** соответствует ID каталога в фиде (`collection id`).
    - **Название каталога** соответствует названию в фиде (`name`).
    - **Ссылка на каталог** соответствует ссылкам на каталоги в фиде (`url`).

**Не формируются объявления страниц каталога, найденных на сайте**

Проверьте, что выполнены условия:

- Сайт соответствует [требованиям](ru/unified-performance-campaign/structure-of-site).
- На сайте есть каталоги, которые соответствуют условиям фильтров:
    - **Название каталога** соответствует названию страницы каталога на сайте.
    - **Ссылка на каталог** соответствует ссылкам на каталоги сайта.
