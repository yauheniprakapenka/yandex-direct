# Настройка бюджета в стратегиях

- [Вопросы и ответы](#вопросы-и-ответы)

В зависимости от стратегии в кампании можно настроить различные бюджеты:

| Стратегия | Тип бюджета |
|---|---|
| Максимум кликов с ручными ставками | [Дневной бюджет](ru/strategies/day-budget) |
| Максимум конверсий / Максимум кликов / Максимум прибыли / Пакетные стратегии | [Недельный бюджет](ru/strategies/week-budget) — для длительных кампаний без точной даты окончания. [Периодный бюджет](ru/strategies/period-budget) — для кампаний, которые имеют заранее определенный бюджет и даты начала и окончания. |

## Вопросы и ответы

**Почему кампания не расходует весь установленный бюджет?**

Возможные причины:

- Низкочастотные ключевые фразы — объявления редко показываются по заданным запросам.
- Низкая максимальная ставка — объявление может показываться только на низких позициях.
- Низкая цена конверсии — система может не найти конверсии по указанной цене.
- Недостаточный бюджет — система назначает ставки не более 10% от недельного бюджета, при низких ставках объявление может показываться на низких позициях.

Подробнее о [причинах, по которым кампания получает мало конверсий](ru/troubleshooting/conversions).

**При каких изменениях в настройках возможен перерасход бюджета?**

Для **дневного** бюджета: изменение бюджета.

Для **недельного** бюджета: изменение бюджета; изменение модели оплаты; перенос даты начала кампании в будущее.

Для **периодного** бюджета: изменение модели оплаты в первый день работы стратегии.

**Почему дневной бюджет расходуется слишком быстро?**

Возможные причины: высокочастотные ключевые фразы; высокие ставки; неограниченный регион показа.

Что можно сделать: увеличить размер дневного бюджета; перейти на автоматическую стратегию; уточнить регион показа; уменьшить ставки.
