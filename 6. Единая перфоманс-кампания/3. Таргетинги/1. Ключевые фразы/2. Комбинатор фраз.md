# Комбинатор фраз

Инструмент подбора фраз помогает расширить набор ключевых фраз и минус-фраз на группу объявлений, используя информацию о поисковых запросах пользователей Яндекса из сервиса [wordstat-2.yandex.ru](https://wordstat-2.yandex.ru/).

Для кампании [Медийные баннеры](ru/products-cpm-campaign/about) доступно расширение только набора ключевых фраз на группу объявлений.

Для перехода к подбору ключевых фраз в меню слева нажмите **Добавить** → **Ключевые фразы**, выберите кампанию и группу, нажмите **Добавить**.

## Шаг 1. Добавьте стартовые фразы

Выберите способ формирования стартовых фраз.

**Комбинатор фраз**

Комбинатор фраз позволяет сформировать список стартовых фраз из слов, по которым вы бы хотели показывать объявления.

Разверните блок **Комбинатор ключевых фраз**, нажав ![Развернуть](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/open-words.png) справа, и укажите слова для генерации — по одному слову в каждой строке. Мы рекомендуем сгруппировать слова, например, в первой группе указать названия товаров, во второй — действия, «продающие» слова, в третьей — модели или марки.

![Комбинатор фраз](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/combination.png)

Можно создать до 7 групп и сгенерировать стартовые фразы длиной до 7 слов. Используйте `*` для пропуска группы — это позволит скомбинировать фразы без слов этой группы и увеличит количество сгенерированных фраз.

> **Примечание**
>
> При создании комбинаций игнорируются операторы и пробелы.

Нажмите **Создать комбинации**. Все полученные стартовые запросы будут добавлены в колонку **Фразы**. Если необходимо, исключите фразы, по которым не прогнозируются показы.

**Ручной ввод**

Введите запрос или несколько запросов, по которым вы хотели бы показывать объявления, через запятую или с помощью **оператора ( | )**.

### Как работает оператор `( | )`

При составлении списка ключевых фраз можно использовать определенный алгоритм, например [список товаров] + [список продающих слов]: *путевка в космос купить*, *путевка в космос недорого*, *тур на Марс купить*, *тур на Марс недорого* и т. д. В этом случае удобно использовать оператор `( | )`.

```
тур +на Марс ( купить|недорого|скидки|акция|цена )
```

Результат: тур +на Марс купить, тур +на Марс недорого, тур +на Марс скидки, тур +на Марс акция, тур +на Марс цена.

```
(купить|подбор|бронирование) тур (+в космос|+на Луну|+на Марс|Альфа Центавра)
```

Результат: купить тур +в космос, купить тур +на Луну, купить тур +на Марс, купить тур Альфа Центавра, подбор тур +в космос, подбор тур +на Луну, подбор тур +на Марс, подбор тур Альфа Центавра, бронирование тур +в космос, бронирование тур +на Луну, бронирование тур +на Марс, бронирование тур Альфа Центавра.

Нажмите кнопку **Добавить**.

## Шаг 2. Проверьте стартовые фразы

В столбце **Фразы** проработайте стартовые запросы.

- Проверьте вложенные запросы по каждой фразе, нажав ![Развернуть](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/open-words.png).

  Для запросов отображается прогноз количества показов в месяц в регионе, указанном в настройках группы объявлений. При составлении прогноза система использует данные за последние 30 дней. Данные считаются только на поиске Яндекса, без учета поисковых площадок Рекламной сети.

- Добавьте в минус-фразы нерелевантные слова из вложенных запросов. Кликните на нужное слово и нажмите **Добавить в минус-фразы**.

  ![Добавление в минус-фразы](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/add-negativ-keywords.png)

  Вложенные запросы с минус-фразами сразу будут удалены из списка стартовых фраз.

  > **Примечание**
  >
  > Минус-фразы не доступны для кампании Медийные баннеры.

- Отметьте фразу или запрос и используйте всплывающие кнопки справа:

  - ![Добавить](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/plus-add-keywords.png) — чтобы перенести фразу или запрос в список ключевых фраз, которые нужно добавить к группе объявлений.
  - ![В минус-фразы](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/minus-add-keywords.png) — чтобы перенести запрос в [минус-фразы](ru/keywords/negative-keywords);
  - ![Удалить](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/basket-add-keywords.png) — чтобы удалить запрос из списка (вместе с вложенными).

  Если в ключевые фразы надо добавить только стартовую фразу, без вложенных запросов, сверните запросы, отметьте фразу и нажмите ![Добавить](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/plus-add-keywords.png) около нее.

Чтобы перенести в список ключевых фраз и стартовую фразу, и вложенные запросы, разверните запросы по фразе, нажав ![Развернуть](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/open-words.png). Отметьте фразу — вложенные запросы будут отмечены автоматически. Нажмите ![Добавить](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/plus-add-keywords.png) около фразы или кнопку **Действие** → **Добавить в ключевые фразы**.

Чтобы удалить списки минус-фраз или ключевых фраз, нажмите ![Настройки](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/settings.png) → **Очистить список**.

## Шаг 3. Добавьте ключевые фразы и минус-фразы к группе объявлений

В столбце **Ключевые фразы** проверьте добавленные фразы. Чтобы уточнить их, нажмите ![Редактировать](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/edit.png). Отметьте минус-слова в предложенном списке. Если требуется, в строке редактирования фразы добавьте [операторы](ru/keywords/symbols-and-operators).

Проверьте также список минус-фраз. Чтобы уточнить минус-фразу, нажмите ![Редактировать](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/edit.png).

Если вы подбираете фразы для группы объявлений кампании Медийные баннеры, назначьте максимальную ставку для новых и измененных условий показа.

Нажмите **Добавить к группе объявлений**.

К ключевым фразам и минус-фразам, которые уже указаны для группы, будут добавлены выбранные ключевые фразы и минус-фразы.
