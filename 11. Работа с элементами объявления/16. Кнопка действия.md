# Кнопка действия

> **Внимание**
>
> С 30 июня текстово-графические объявления в Единых перфоманс-кампаниях станут доступны только для редактирования. Для задач, которые раньше решались с помощью текстово-графических объявлений, можно будет создавать комбинаторные. Подробнее — на странице [Обновление текстово-графических объявлений до комбинаторных](ru/unified-performance-campaign/upgrade-to-comb).

Кнопка действия добавляется в [видеообъявления](ru/products-cpm-campaign-video/about) медийных кампаний или текстово-графические объявления в Единой перфоманс-кампании с изображением или видео. Она привлекает внимание и побуждает пользователей взаимодействовать с вашим объявлением. Кнопка дополняет объявление еще одной ссылкой. По клику на текст или видео пользователь переходит на одну страницу сайта, а по кнопке действия — на другую. Ссылки также могут совпадать.

## Как настроить

По умолчанию кнопка действия настраивается автоматически — нейтральный текст, например, «Узнать больше», «Перейти на сайт», и ссылка, совпадающая с ссылкой объявления.

Вы можете выбрать более продающий текст для кнопки и указать другую ссылку, когда создаете или редактируете объявление. Для этого в блоке **Кнопка** вместо текста по умолчанию выберите подходящий и укажите ссылку на страницу перехода. Она может отличаться от ссылки в объявлении. Чтобы добавить ссылку на мессенджер, используйте форматы:

- для Telegram — `t.me/username`;
- для WhatsApp — `wa.me/phone_number`.

![Настройка кнопки действия](https://cdn-viewer.diplodoc.com/docs-assets/support-direct-pro/rev/r20128576/ru/_assets/button-your-text.png)

Добавить кнопку в несколько текстово-графических объявлений в Единой перфоманс кампании можно с помощью [массовых действий](ru/campaigns/multi-edit).

Статистика показов объявлений с дополнительным оформлением доступна в [Мастере отчетов](ru/statistics/new-reports-wizard). Чтобы проанализировать количество кликов по кнопке, добавьте в отчет группировку **Место клика** → **Кнопка**.
